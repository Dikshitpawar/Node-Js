Q1. What is Node.js?

Node.js is a JavaScript runtime environment used to run JavaScript outside the browser.

Q2. Is Node.js a programming language?

No. It is a runtime environment.

Q3. Why is Node.js fast?

Because it uses Chrome’s V8 engine and non-blocking architecture.

Q4. Can we use DOM in Node.js?
No.
Because DOM browser ka part hai.

4) V8 Engine Kya Hai?

V8 ek JavaScript engine hai
jo JavaScript code ko machine code me convert karta hai.
Originally:
✅ Chrome browser me

Baad me:
✅ Node.js me bhi use hua
Q1. What is V8 Engine?

V8 is Google’s JavaScript engine that converts JavaScript into machine code.

Q2. Why Node.js is fast?

Because it uses the V8 engine and non-blocking architecture.

Q3. What is JIT Compilation?

Just-In-Time compilation converts code into machine code during runtime.

Q4. Who developed V8?

Google.

Q5. Does V8 execute JavaScript line by line?

Not exactly.

It uses interpreter + compiler combination.

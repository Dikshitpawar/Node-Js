**Built-in middleware

1) express.json() :- parse the json data
2) express.urlencoded({extended:true}) :- parse the content come from html forms
3) express.static() :- used for serve the static files
4) express.text() :- parse the text data
5) express.Router() :- one mini express application



**Third-Party middleware

1) morgan :- npm i morgan
Logs HTTP requests
Useful for debugging

Example:
const morgan = require('morgan');
app.use(morgan('dev'));

2)CORS :
Allows requests from different domains
Solves Cross-Origin Resource Sharing issues

Example:
const cors = require('cors');
app.use(cors());

3)Helmet :
Adds security-related HTTP headers
Helps protect against common web vulnerabilities

Example :
const helmet = require('helmet');
app.use(helmet());

4)Cookie Parser :
Reads cookies sent by the browser

Example:
const cookieParser = require('cookie-parser');
app.use(cookieParser());

5)Body-Parser :
Parses incoming request bodies 

Example:
const parser = require('body-parser');
app.use(parser.json());
